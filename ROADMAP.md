# Roadmap

The current release covers local import, audio/video libraries, full + mini player with repeat/shuffle/speed, playlists (reorder, delete, resume per item), folders, and persistence.

## Planned phases

- **Phase 2 — Lock screen / Control Center controls:** `MPNowPlayingInfoCenter` + `MPRemoteCommandCenter`.
- **Phase 3 — Folders tab:** promote Folders from Settings into the main `TabView`.
- **Phase 4 — URL import module:** lawful direct MP3/MP4 URLs and HLS (behind the existing placeholder).
- **Phase 5 — iCloud sync:** CloudKit-backed SwiftData container.
- **Phase 6 — Widgets:** Home Screen / Lock Screen widgets.
- **Phase 7 — AirPlay / external display.**
- **Phase 8 — macOS Catalyst port.**
